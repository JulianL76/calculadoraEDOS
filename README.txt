DJ3 Calculator

El ejecutable se encuentra en la carpeta del proyecto de neatbeans "CalculadoraEDOS" dentro de esa carpeta se diríjase a la carpeta "dist", ahí encontrara un archivo "calculadoraEDOS.jar" el cuál será el ejecutable

El ejecutable se encuentra en la ruta:
"\CalculadoraEDOS\dist\CalculadoraEDOS.jar"

Se puede abrir mediante el acceso directo "CalculadoraEDOS ejecutable" dentro de esta misma
carpeta
Se anexa manual de usuario, y el proyecto de neatbeans con el codigo fuente
